# Network-Analyzer
Refreshing my knowledge on Python.

### What it does

- Perform internet speed tests using Ookla's Speedtest
- Monitor network latency to Google DNS (8.8.8.8) or Cloudflare DNS (1.1.1.1)
- Track average ping over time
- Track average packet loss
- Generate alerts when network latency rises significantly above normal levels
- Detect packet loss events

### Features

- Automatic server selection
- Download speed testing
- Upload speed testing
- Ping measurement
- Continuous monitoring mode
- Current ping statistics
- Historical ping averaging
- Packet loss monitoring
- High-latency alerting

### Requirements
- speedtest-cli
- icmplib

### Install dependencies:
```
pip install -r requirements.txt
```

### Purpose

This project was created as a learning exercise to develop practical skills in:
- Python programming
- Network troubleshooting
- Network performance monitoring
- Data collection and analysis
- Working with third-party Python libraries
